# A simple XML feed aggregator made for [10k Apart Challenge](https://a-k-apart.com)

As part of the [10k Apart Challenge](https://a-k-apart.com) I built this simple XML feed aggrigator where the initial payload is less than 10kb. The aggregator works even without JavaScript by rendering 10 feed items on server side. For more rules, please refer: [ttps://a-k-apart.com/faq].
